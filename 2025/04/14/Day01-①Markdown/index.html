Markdown学习

标题

‘###’ + ‘空格’ + ‘内容’为三级标题

‘####’ + ‘空格’ + ‘内容’为四级标题

字体

*Hello World! ** 加粗
Hello World! 斜体
Hello World! 加粗+斜体
Hello World! ~~删除线~~*

引用

‘>’ + ‘空格’ 表示引用

分割线

‘—’ 、‘***’ 是分割线

图片

‘!’ + ‘[图片名字]’ + ‘(图片网络链接or本地图片)’

超链接

‘[超链接名称]’ + ‘(网站)’
点击跳转到教程

列表

有序列表：’1.’ + ‘空格’ 按回车会自动生成下一条 双击回车退出列表

  1. A
  2. B
  3. C

无序列表：’-‘ + ‘空格’

  • A
  • B
  • C

表格

右键可以直接插入简单表格 点击查看源代码模式可以知道表格的格式

名字 性别 生日
Tom 男 2020.2.20

|名字|性别|生日|
上一行回车即可生成以上述内容为表头的表格

代码

‘ ``` ‘ (Tab上边那个点) + ‘编程语言’
