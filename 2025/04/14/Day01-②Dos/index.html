基本的Dos命令

开启DOS控制台的方式

普通打开

  1. win + R → cmd → 回车
  2. 菜单 → 命令提示符
  3. 在文件夹内按住 shift + 鼠标右键 → 在终端打开
  4. 资源管理器的地址栏前面加上：cmd 路径

管理员方式运行

菜单 → 右键命令提示符 → 以管理员身份运行
